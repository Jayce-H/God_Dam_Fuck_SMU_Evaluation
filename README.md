**重点是评价数超过50%的加3分**  
这大概就是这破学校最后的挣扎了, 虽然我不甚好学

![看图](./png.png)


为了3分的成绩, 我是如此的不务正业

# 01 支持特性:  
- 1/3的概率出现95分评价, 躲避教务系统追踪, 1/9概率出现90分评价
- 自定义主观题评价内容, 随机抽取, 同上是为了躲避系统追踪
- 打包exe文件, 去除python环境依赖
